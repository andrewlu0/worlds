# worlds

Spotify Music Visualizer using React, Three.js and react-three-fiber

To run locally:
```
yarn install
```
```
yarn dev
```

Perlin noise from:
GLSL textureless classic 3D noise "cnoise",
with an RSL-style periodic variant "pnoise".
